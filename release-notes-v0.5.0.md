# Trench Coat v0.5.0

Legal-first multi-hop privacy cloak with Tor-aware CLI and cyberpunk control nexus.

## Install

```bash
git clone https://github.com/Pitchfork-and-Torch/trench-coat.git
cd trench-coat
python -m venv .venv
pip install -e .
trench up --accept-legal --wait-tor 60
```
